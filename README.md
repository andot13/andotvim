Basics

`;;` switch modes

`,` as leader key

`Shift + k` Move selected a line going top
`Shift + j` Move selected a line going bottom

Window Movements

`ctrl + h` Move window left 
`ctrl + j` Move window down 
`ctrl + k` Move window up 
`ctrl + l` Move window right 

Multiple Select

`ctrl + n` Select
